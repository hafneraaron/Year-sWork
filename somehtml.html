CSS Tutorial:
Internal Style Sheet

Each HTML document may have only one internal style sheet but may also link to external style sheets. We will learn about external style sheets in the next lesson.

Color Values

Color may be expressed as a named color or as an RGB value preceded by the octothorp (pound sign,#). No quotations are used for CSS color property values.
